# 自动化点击工具
## 使用步骤
输入要发送的内容和发送次数；
点击`记录位置1`，1s内把鼠标移到聊天输入框；
`记录位置2`，1s内移动到发送按钮的位置；
点击`确认位置`后，再点击`开始执行`

## 原理
界面基于Tkinter库，通过模拟鼠标指针与键盘行为进行输入与点击操作。暂不支持中文输入。
